# PyStormTracker-Data

This repository hosts test datasets used by [PyStormTracker](https://github.com/mwyau/PyStormTracker).

## Datasets

### ERA5 MSL and VO850 (Dec 2025 - Feb 2026)
This dataset includes Mean Sea Level Pressure (MSL) and 850 hPa Vorticity (VO) data from the ERA5 reanalysis for the period of December 2025 to February 2026 (DJF).

- **Variables:** `msl`, `vo`
- **Resolutions:** 
  - `0.25x0.25` degrees (High resolution)
  - `2.5x2.5` degrees (Coarse resolution)
- **Source:** [ERA5 Reanalysis](https://www.ecmwf.int/en/forecasts/dataset/ecmwf-reanalysis-v5)
- **Format:** NetCDF4 (.nc)
